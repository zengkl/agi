\begin{definition}[The dual linear program]
Given $\max\{c^Tx: x\in \R^n, Ax\leq b\}$
the {\em dual linear program} is

$$ \min b^Ty  $$
\begin{eqnarray*}
A^Ty &=& c  \\
y &\geq& 0 
\end{eqnarray*}
